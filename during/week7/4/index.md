## Wk 7 - Thursday - jQuery in Detail

* Three things to pass to `$()`
  * CSS Selectors
  * DOM Elements
  * Functions
* Searching
  * `.first()`
  * `.last()`
  * `.length`
  * `.find("css-selector")`
  * `.children()`
  * `.parent()`
  * `.closest()`
* HTML Getters
  * `.val()`
  * `.attr("class")`
  * `.style("color")`
  * `.prop("hidden")`
  * `.html()`
* Setters each take one additional parameter on getters.
* DOM Manipulation
  * `.hide()`
  * `.show()`
  * `.clone()`
  * `.remove()`
  * `.append(element)`
* Event Handlers
  * `.on("click", function(){})`
* Communication
  * `$.ajax()`
* JavaScript MVC Framework Overview
